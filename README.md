# Painel 4G Installer


System Suported

Ubuntu 14<br>
Debian 8

Install script 

apt-get update && apt-get upgrade -y && apt install dos2unix -y && wget https://raw.githubusercontent.com/flindao/Painel-4G-Installer/main/Shell%20Script/int4g.sh && dos2unix int4g.sh && chmod 777 int4g.sh && ./int4g.sh
